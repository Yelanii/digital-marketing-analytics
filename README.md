# Daraz E-Commerce Usability Analysis

## Overview

This project presents a usability evaluation of the Daraz.lk e-commerce platform using UX research methods, usability testing, and statistical analysis techniques.

The study investigated how different interface conditions affect:

* Product search effectiveness
* User confidence
* Task efficiency
* Memorability
* User satisfaction and trust

The evaluation used hypothesis-driven usability testing and quantitative analysis methods to identify usability patterns and provide evidence-based UX recommendations for improving the online shopping experience.

---

# Project Objectives

The main objectives of this project were to:

* Evaluate the usability of the Daraz.lk platform
* Analyse user interaction behaviour during product search tasks
* Measure the impact of filters on product search effectiveness
* Evaluate user memorability in repeated search tasks
* Examine the relationship between product ratings and user trust
* Apply statistical analysis techniques to usability testing data
* Provide UX improvement recommendations based on findings

---

# Research Areas

The project evaluated four major usability metrics:

## 1. Efficiency

Evaluated whether experienced users complete the checkout process faster than inexperienced users using an independent-sample t-test.

## 2. Effectiveness

Evaluated whether applying filters improves users’ ability to find products more effectively using a paired-sample t-test.

## 3. Memorability

Evaluated whether repeated exposure improves task performance using repeated search tasks and ANOVA analysis.

## 4. Satisfaction and Trust

Evaluated the relationship between product ratings and user trust using Pearson correlation analysis.

---

# Methodology

The usability evaluation used moderated remote usability testing methods.

The study included:

* Task-based usability testing
* User observation
* Screen recording analysis
* Quantitative and qualitative data collection
* Statistical usability analysis
* Hypothesis testing
* Behavioral analysis

Participants completed realistic e-commerce tasks on the Daraz.lk platform under controlled usability testing conditions.

---

# Statistical Analysis Techniques

The project applied multiple statistical analysis methods, including:

* Independent-Sample t-Test
* Paired-Sample t-Test
* ANOVA
* Pearson Correlation Analysis
* Descriptive Statistics
* Likert Scale Analysis
* Comparative Usability Evaluation

---

# Key Findings

The usability evaluation identified several important findings:

* Filtering features improved product search effectiveness
* Experienced users completed checkout tasks more efficiently
* Repeated exposure improved task memorability and confidence
* Higher product ratings increased user trust and satisfaction
* Interface design significantly influenced user interaction behaviour

The findings demonstrate how usability testing and UX analytics can support evidence-based improvements in e-commerce platforms.
